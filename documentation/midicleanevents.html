MidiCleanEvents

DWORD WINAPI MidiCleanEvents
(
	DWORD dwEvents, 
	LPBRELS_MIDI_EVENT lpEvents
);

This function deallocates an array created by MidiGetTrackEvents.

Parameters

  • dwEvents: number of events in the array. Use the value returned by MidiGetTrackEvents.
  • lpEvents: pointer to the array.

Return values

Returns the number of cleaned events (the same value in dwEvents parameter) or zero if an error occurred.

Remarks

Always call MidiCleanEvents after the array created by MidiGetTrackEvents is not necessary anymore.

Example

For an example, see MidiGetTrackEvents.
