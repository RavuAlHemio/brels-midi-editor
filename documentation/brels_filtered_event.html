BRELS_FILTERED_EVENT

typedef struct
{
	QWORD qwFilter;
	QWORD qwStart;
	QWORD qwEnd;
} BRELS_FILTERED_EVENT, *LPBRELS_FILTERED_EVENT;

This structure is used to store indexes of events filtered by the function MidiFilterTrackEvents.

Members

  • qwFilter: filter that applies to this event. When filtering more than one type of event, refer to this member to know what kind of event was retrieved.
  • qwStart: some events are composited, for instance notes. This corresponds to the first part of the event (the note on event, or 0x9n)
  • qwLast: in note events, this is the index of the last part of the event (the note off event, or 0x8n). In simple events, it has the same value than qwStart

Remarks

This structure only stores event indexes. To access actual event data, use MidiEventGet or MidiGetTrackEvents.
