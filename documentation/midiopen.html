MidiOpen

BOOL WINAPI MidiOpen
(
	LPSTR lpstrFile,
	DWORD dwDevice,
	LPHANDLE lphSequence
);

This function opens a MIDI file.

Parameters

  • lpstrFile: pointer to a null-terminated string containing the full name and path of the MIDI file to open.
  • dwDevice: ID of the midiOut device to use for playback. Use DEFAULT_DEVICE ((DWORD) -1) for the MIDI Mapper.
  • lphSequence: pointer to a variable to receive the handle of the open sequence.

Return values

TRUE if the sequence was open or FALSE if not. The possible reasons for an error are a bad file name, a busy midiOut device or an invalid pointer for the handle.

Remarks

Call MidiClose to close the sequence and free the resources associated to it. Call MidiPlay to listen to the sequence.

Example

HANDLE Sequence;

// Open the sequence
MidiOpen("test1.mid", DEFAULT_DEVICE, &Sequence);

// Play it
MidiPlay(Sequence);

// Close it anyway
MidiClose(Sequence);
